Summary

The purpose of this project was to analyze how weather changes as you get closer to the equator. To accomplish this analysis the OpenWeatherMap API was employed to gather data from over 500 Cities.

After assembling the dataset, Matplotlib was used to plot the various aspects of weather as they related to latitude. In particular the relationship between latitude and temperature, cloudiness, wind speed, and humidty was examined. This site contains the visualizations that were created as part of the analysis. The plots created were as follows:

  • Temperature (F) vs. Latitude
  • Humidity (%) vs. Latitude
  • Cloudiness (%) vs. Latitude
  • Wind Speed (mph) vs. Latitude
